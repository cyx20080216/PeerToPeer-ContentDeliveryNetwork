# Simple-ContentDeliveryNetwork
It is a simple CDN.

It requires [fsnotify](https://github.com/fsnotify/fsnotify)
# Use build script
```bash
sh build.sh
```
# Build center
```bash
cd DataAndRegistrationCenter/
go get
go build
```
# Build node
```bash
cd Node/
go get
go build
```
# Build router
```bash
cd Router/
go get
go build
```
